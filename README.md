# Real Estate Management System (REMS)
## 📋 Project Overview

The Real Estate Management System (REMS) will be a web platform to streamline the processes of tenants, and operators. Features include automated invoices, secure authentication, and clear separation of tenant, clerk, and admin tasks (refer to the project description for system requirements). With the objective to reduce manual workload, provide 24/7 resources to both tenants and operators and a clear paper trail for all the users. 


### 🎯 Core Objectives

- Provide an online search for available units.
- Appointment Scheduling System: To prevent booking conflicts for viewings and giving leasing agents a calendar view of appointments.
-  Automate lease creation and invoice generation.
-  Track rent payments and maintenance requests.
-  Support secure login and role-based access.
